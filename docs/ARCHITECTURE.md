# 🛠️ Arquitectura de Proyecto

## Stack Tecnológico Recomendado
* **Frontend:** Next.js 14+ (App Router).
* **Lenguaje:** TypeScript.
* **Estilos:** Tailwind CSS.
* **Despliegue:** Vercel / Cloudflare.

## Estructura de Páginas (Sitemap)
1.  **Home:** El núcleo tecnológico.
2.  **Servicios:** Catálogo integral.
3.  **Portafolio:** Casos de éxito y proyectos.
4.  **Nosotros:** Historia y equipo.
5.  **Blog:** Recursos y conocimiento técnico.
6.  **Contacto:** Captación de proyectos.

## Componentes Globales
* `Navbar`: Glassmorphism persistente.
* `Footer`: Mapa de sitio y contacto corporativo.
* `CTA_Section`: Botón verde neón sobre fondo oscuro.
