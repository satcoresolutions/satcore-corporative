# 🎨 Guía de Estilo Visual (UI/UX)

## Paleta de Colores
### Primarios
* **Azul Sat:** `#132A8E` (Branding y Títulos)
* **Verde Core:** `#1BFF3C` (CTA y Energía)

### Neutros & Premium
* **Negro Suave:** `#0A0A0A` (Fondos Premium)
* **Gris Claro:** `#F6F9FC` (Cards y UI Limpia)
* **Blanco:** `#FFFFFF` (Texto sobre oscuro)

## Regla de Oro (70/20/10)
* **70% Neutros:** Fondos y áreas de descanso.
* **20% Azul:** Identidad y estructura.
* **10% Verde Neón:** "Nitro" para llamadas a la acción. **No saturar.**

## Tipografía
1.  **Inter:** Fuente principal para UI y lectura.
2.  **JetBrains Mono:** Uso exclusivo para fragmentos de código y datos técnicos.

## Sistema de Espaciado (Tokens)
* `space-md`: 16px (Padding interno)
* `space-2xl`: 48px (Secciones grandes)
* `space-4xl`: 96px (Espaciado Hero)

## Border Radius
* Botones: `8px`
* Cards: `12px`
* Inputs: `6px`
